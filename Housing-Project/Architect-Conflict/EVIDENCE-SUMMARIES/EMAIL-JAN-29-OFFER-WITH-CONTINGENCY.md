## STRATEGIC SIGNIFICANCE: THE CONTINGENCY CLAUSE

**Critical Quote:**
> "I wrote as well that if the number of the apartment reduces for whatever reason a new offer will need to be made for satisfaction of both parties."

**Why This Matters:**
1. **Architect Acknowledged Uncertainty:** By writing contingency clause, architect explicitly acknowledged apartment count could change
2. **Architect Required New Agreement:** Architect agreed that IF apartments reduce, "new offer will need to be made"—showing architect understood changes require NEW agreements, not unilateral architect action
3. **Original Contract Covers 2-5 Apartments:** Original contract already analyzes 2/3/4/5 apartment options. If apartments ultimately were 3 (reduced from 4), this clause shows architect should have issued NEW offer, not demanded payment for "4-apartment work"
4. **Demolition Rejection = Reduced Scope:** When client rejected demolition, this constituted a material change that triggered the contingency clause—requiring "new offer" per architect's own terms

**This proves:** Even if architect did any additional work on 4-apartment design (which he didn't), the contingency clause would have triggered if apartment count changed, requiring renegotiation per architect's own written agreement.
